version: v0.10.1

plugin management: Lazyvim

![](./nvim.gif)

## Installation

Make backup for your old nvim:

```sh
mv ~/.config/nvim ~/.config/nvimbak
mv ~/.local/state/nvim ~/.local/state/nvimbak
```

install:

```sh
git clone https://github.com/qiancijun/nvim-env.git ~/.config/nvim
```
